## ğŸ’¡ The Image URL Field

In `admin/add_product.php`, you'll see this field:

```
Image URL: [________________________]
```

Just paste any HTTPS image URL there. Examples:
- `https://images.unsplash.com/photo-xxx?w=500`
- `https://i.ibb.co/xxxxx/image.jpg`
- `https://res.cloudinary.com/xxx/image.jpg`

**That's it!** The application will fetch and display the image from that URL.
